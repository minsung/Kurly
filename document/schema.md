# DB Schema

## User (사용자)
- 사용자 정보를 나타내는 엔티티입니다.

| 속성 (Attributes)  | 데이터 타입   | 설명                                   | 제약 조건                    |
|--------------------|---------------|----------------------------------------|------------------------------|
| id                 | Long          | 사용자 고유 식별자                      | 기본 키 (Primary Key)       |
| email              | 문자열        | 사용자 이메일 주소                      | 고유, NULL 불가능            |
| password           | 문자열        | 사용자 비밀번호                         | NULL 불가능                  |
| name               | 문자열        | 사용자 이름                            | NULL 불가능                  |
| mobileNumber       | 문자열        | 사용자 휴대폰 번호                      | NULL 불가능                  |
